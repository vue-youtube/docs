VueYoutube

Vue Plugin powered by Vite and TS

This plugin makes it easy to integrate the YouTube Iframe Player into your Vue 2/3 app.

Get Started
View on GitHub

Vue 2/3 support

Both major versions of Vue are supported with the magic of VueDemi.

Flexible usage

Do it the way you like to do it: Use the plugin as a Composable or Component.

Reactive all the way

Everything is reactive. Even the video ID - you are now able to switch the video by just updating the ref.

Full typing support

The plugin is written in 100% TypeScript and every API is fully typed.
